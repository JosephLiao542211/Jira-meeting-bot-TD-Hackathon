# Guidelines

## Action Items
- Assign tasks when someone says "I will", "I'll handle", "can you take care of"
- Summary should be concise, under 15 words

## Bug Reports  
- Flag bugs when someone says "broken", "not working", "issue with", "failing"
- Always include description with steps if mentioned

## Decisions
- Log decisions when someone says "we decided", "we agreed", "going with"
- Summary should capture the decision clearly
